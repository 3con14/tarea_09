MOBILE FIRST

Como su propio nombre indica mobile first o por su traducción en inglés, “los móviles primero”, resume perfectamente esta filosofía de desarrollo web.

Básicamente es un modo de diseñar sitios web teniendo en cuenta primero, un dispositivo móvil. Diseñamos primero para pantallas reducidas, y tras tener la maqueta preparada, realizar un escalado, es decir, aumentar el tamaño y adaptarlo a una pantalla de escritorio.

¿Qué es Responsive Design?

Responsive Design es una técnica de diseño web que se utiliza para crear un sitio que se adapte y cambie según el dispositivo o pantalla que está utilizando el usuario. No importa si acceden desde una pantalla panorámica, una tablet, un notebook o un smartphone, siempre verán una página web optimizada.
